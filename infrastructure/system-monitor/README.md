# system-monitor MCP Server

🇬🇧 System Monitor Model Context Protocol (MCP) server.
🇰🇷 시스템 모니터링 Model Context Protocol (MCP) 서버입니다.

## 🇬🇧 Setup / 🇰🇷 설정 방법
1. 🇬🇧 Create venv / 🇰🇷 가상환경 생성: `python -m venv venv`
2. 🇬🇧 Activate venv / 🇰🇷 가상환경 활성화: `venv\Scripts\activate`
3. 🇬🇧 Install deps / 🇰🇷 의존성 설치: `pip install -r requirements.txt`

## 🇬🇧 Usage / 🇰🇷 사용법
🇬🇧 Run with inspector / 🇰🇷 인스펙터로 실행:
`npx @modelcontextprotocol/inspector python server.py`
