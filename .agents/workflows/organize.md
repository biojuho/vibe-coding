# /organize — 폴더 정리 워크플로우

## 실행 절차

1. `.agents/rules/project-rules.md`를 읽어 현재 폴더 구조 규칙을 확인한다.

2. 워크스페이스 루트의 전체 파일 트리를 스캔한다.

3. 규칙에 맞지 않는 항목을 찾아 이동 계획을 표로 정리한다:
   | 현재 위치 | 제안 위치 | 이유 |
   |-----------|-----------|------|

4. **사용자에게 승인을 요청한다** — 파일 이동은 승인 없이 절대 실행하지 않는다.

// turbo
5. 승인된 항목에 대해 파일/폴더를 이동한다.

// turbo
6. 이동된 파일의 import 경로, 참조 경로를 검색하여 수정한다.

7. 변경 내용을 Knowledge Base에 기록한다.
