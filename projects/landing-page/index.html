Behind The Scenes

The 3-Layer Architecture

AI의 불확실성을 제어하고 결정론적 결과를 보장하는 혁신적인 설계 구조입니다.

1

지침 레이어 (Directive Layer)

Markdown 기반의 표준 운영 절차(SOP)와 .agents/rules 내 규칙들로 구성됩니다. 비즈니스 로직과 제품의 방향성, 엄격하게 검증되어야 할 목표(Goal)들을 자연어로 정의합니다.

2

오케스트레이션 레이어 (Orchestration Layer)

다양한 AI 에이전트(Antigravity 등)가 지침을 기반으로 최적의 워크플로우 경로를 지능적으로 의사결정하고 결합합니다. 상황에 따른 조건부 분기와 에러 예외 처리를 처리하는 접착제 역할을 합니다.

3

실행 레이어 (Execution Layer)

오케스트레이션에 의해 호출되는 결정론적이고 견고한 실행 엔진입니다. 비즈니스 API 통신, 로컬 DB 트랜잭션, 쉘 스크립트 실행 등을 100% 보장하는 Python/Bash 스크립트로 동작합니다.
