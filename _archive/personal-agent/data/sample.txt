Vibe Coding Project Overview

1. Introduction
Vibe Coding is a project to create an intelligent personal agent.
The agent "Jarvis" provides daily morning briefings including weather and news.
It uses Google Gemini (or OpenAI) as the intelligence engine.

2. Architecture
- Language: Python
- Framework: LangChain (Planned)
- Database: ChromaDB (Vector Store)
- Interface: CLI (Current), Streamlit (Future)

3. Key Features
- Auto-Briefing: Weather (wttr.in) + News (RSS)
- RAG: Document search using local PDFs/TXTs
- Mock Mode: Simulation when API quota is exceeded.
