Vibe Coding Project Knowledge Base

1. What is Vibe Coding?
"Vibe Coding" is a philosophy and methodology for AI-assisted software development.
It emphasizes the synergy between human creativity and AI productivity.
Instead of just writing code, the developer "vibes" with the AI, guiding it through high-level intent and creative direction.
The goal is to produce high-quality, aesthetically pleasing, and functional software at a rapid pace.

2. Core Principles of Vibe Coding
- Aesthetics First: Software should not only work well but look and feel premium.
- Synergy: AI handles the heavy lifting (boilerplate, refactoring, testing), while the human provides the soul (design, architecture, decision-making).
- Speed & Iteration: Rapid prototyping and continuous refinement.

3. The "Personal Agent" Project
Personal Agent (codenamed "Jarvis") is a flagship project within the Vibe Coding ecosystem.
It serves as an intelligent assistant for the developer, automating daily routines and providing information.

4. Personal Agent Capabilities
- Morning Briefing: Compiles weather (using wttr.in) and top news headlines (RSS feeds) into a concise summary.
- RAG (Retrieval-Augmented Generation): Can answer questions about the user's specific projects and documents by indexing local files.
- Smart Query: Uses LLMs (OpenAI GPT-4o or Google Gemini 1.5 Pro) to understand complex natural language queries.

5. Technical Stack
- Language: Python 3.10+
- LLM Integration: OpenAI API / Google Generative AI SDK
- Vector Database: Simple local vector store (using scikit-learn & numpy) for lightweight RAG.
- Interface: Currently CLI-based, with plans for a Streamlit web UI.
